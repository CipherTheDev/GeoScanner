Pure Python GeoIP API
=====================

This library is based on `Maxmind's GeoIP C API <https://github.com/maxmind/geoip-api-c>`__.

Tested with Python version 2.6, 2.7, 3.2 and 3.3.

Installation
------------

You can easily install pygeoip from PyPi.

.. code:: python

    pip install GeoIP
    easy_install GeoIP
